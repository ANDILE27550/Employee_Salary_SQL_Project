Employee_Salary_SQL_Project

Project Description :
This project analyzes employee salary data using SQL. The goal is to understand salary distribution across departments and categorize employees based on salary levels.

Tools Used :
MySQL 8.0 Command Line Client
Database Structure
Table: employees 
Columns:
employee_id
name
department
salary

SQL Skills Demonstrated :

CREATE DATABASE and TABLE
INSERT statements
GROUP BY and AVG
ORDER BY
CASE statements for salary categorization

Key Insights :

Average salary per department was calculated.
Highest and lowest paid employees were identified.
Employees were grouped into Low, Medium, and High salary levels.

Outcome :

This project demonstrates the ability to write clean SQL queries for business analysis and reporting, suitable for entry-level data and SQL freelancing work.
